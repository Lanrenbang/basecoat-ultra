Welcome to Basecoat Ultra

A Tailwind-first UI component library designed for high performance, minimalism, and deep customization.

Framework Agnostic

Works seamlessly with Vanilla HTML/JS, React, Vue, Svelte, or any other framework.

Tailwind-First

Built on top of Tailwind CSS with a utility-first approach for maximum flexibility.

Theme Support

Built-in Catppuccin theme suite with dark/light mode support and customizable accents.

Getting Started

Acknowledgment: Basecoat Ultra is built upon the excellent foundation of Basecoat by hunvreus. We've enhanced it with modularity, Catppuccin themes, and Svelte 5 support while maintaining its core philosophy.

Basecoat Ultra provides two main packages to suit different development needs:

Ultra Package Installation

Package Manager

bun add @lanrenbang/basecoat-ultra

Also supports npm or pnpm

CSS Setup (Bundlers)

/* In your CSS file (e.g., style.css) */
@import "tailwindcss";
@import "@lanrenbang/basecoat-ultra";
/* Optional external components */
@import "@lanrenbang/basecoat-ultra/datepicker.css";
@import "@lanrenbang/basecoat-ultra/resizable.css";
/* Optional theme */
@import "@lanrenbang/basecoat-ultra/theme/catppuccin";

JavaScript Setup

// Option A: Import all (recommended)
import '@lanrenbang/basecoat-ultra/all';
// Option B: Cherry-pick (core first)
import '@lanrenbang/basecoat-ultra/basecoat';
import '@lanrenbang/basecoat-ultra/tabs';
// External components (separate)
import '@lanrenbang/basecoat-ultra/datepicker';
import '@lanrenbang/basecoat-ultra/resizable';

CDN Usage (No Build Tool)

<!-- CSS -->
<link href="https://cdn.jsdelivr.net/npm/@lanrenbang/basecoat-ultra@latest/dist/css/basecoat.cdn.min.css" rel="stylesheet">
<!-- JavaScript -->
<script src="https://cdn.jsdelivr.net/npm/@lanrenbang/basecoat-ultra@latest/dist/js/all.min.js" defer></script>

Anti-Flash Script

To prevent page flicker (FOUC) when using the Catppuccin Theme Switcher, add this script at the top of your HTML <head> tag:

<script>
(function() {
  try {
    const t = localStorage.getItem('catppuccin-theme');
    const a = localStorage.getItem('catppuccin-accent');
    const m = localStorage.getItem('basecoat-mode');
    const r = document.documentElement;
    if (t && a) {
      r.classList.add(`theme-${t}`, `accent-${a}`);
      if(['frappe','macchiato','mocha'].includes(t)) r.classList.add('dark');
    } else if (m) {
      r.classList.toggle('dark', m === 'dark');
    } else {
      r.classList.toggle('dark', window.matchMedia('(prefers-color-scheme: dark)').matches);
    }
  } catch(e) {}
})();
</script>
